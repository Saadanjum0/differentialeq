Common Examples

First-Order Linear

DE: y' + 2*y = sin(x)

Solution: y = e^(-2*x) * (C + 0.5*sin(x) - 0.5*cos(x))

Second-Order Linear

DE: y'' + 4*y = 0

Solution: y = C1*cos(2*x) + C2*sin(2*x)

Separable Equation

DE: y' = y^2 * sin(x)

Solution: y = 1/(C - cos(x))

Bernoulli Equation

DE: y' + y = y^2 * e^x

Solution: y = 1/(C*e^x - 1)
